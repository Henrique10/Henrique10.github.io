    HTML (abreviação para a expressão inglesa HyperText Markup Language, que significa Linguagem de Marcação de Hipertexto) É uma linguagem de marcação utilizada para produzir páginas na Web criada na década de 1990 por Tim Berners Lee. A tecnologia é fruto da junção entre os padrões HyTime e SGML.

    HyTime é um padrão para a representação estruturada de hipermídia e conteúdo baseado em tempo. Um documento é visto como um conjunto de eventos concorrentes dependentes de tempo (como áudio, vídeo, etc.), conectados por hiperligações. O padrão é independente de outros padrães de processamento de texto em geral.

    SGML é um padrão de formatação de textos. Não foi desenvolvido para hipertexto, mas tornou-se conveniente para transformar documentos em hiper-objetos e para descrever as ligações.

    O HTML trabalha com o sistema de tags.
    Ex: 

                    <tag>Conteúdo da tag</tag>

    Obs: Toda tag aberta deve ser fechada.

    Uma tag HTML pode conter outra tag dentro dela, e quantas forem necessárias.
    Ex: 

                    <tag>
                     <outraTag>texto</outraTag>
                </tag>



    HTML - <html> </html>

    Essa tag é responsável por definir o início de um documento HTML e seu fim. Logo deve ser aberta no início do documento e fechado no seu final.

    HEAD - <head></head>

    Tag que define o início e o fim do cabeçalho do documento.

    BODY - <body></body>

    Tag onde devem ser inseridos os conteúdos da página que deverão ficar visíveis ao usuários quando a página for renderizada no navegador.

    Apenas com essas tags já podemos criar um página básica em HTML.

    Ex: 
                        <html>

                        <head>

                            <title>Título da página</title>


                        </head>
                        
                        <body>

                           Conteúdo da página
    
                        </body>

                    </html>

    Obs: Para que seus codigos fiquem acentuados de maneira corretta é recomendado inserir no inicio logo após o inicio do cabeçalho(<head>) a tag <meta charset="utf-8">
